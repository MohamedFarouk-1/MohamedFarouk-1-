## What I'm Building

### 🤖 [AgentPay](https://github.com/MohamedFarouk-1/agentpay)
On-chain payment infrastructure for autonomous AI agents. USDC vault with per-agent spending limits, deployed on Base + Arbitrum. Full stack: Solidity → FastAPI → React + wagmi. Built to test whether on-chain authorization could replace human approval loops for agent transactions.

### ⚡ [SparseOpt](https://github.com/MohamedFarouk-1/SparseOpt)
PyTorch computation graph optimizer using `torch.fx`. Automates operator fusion, dead node elimination, and dropout removal at inference time — with zero changes to model code. Benchmarked 4% latency reduction on BERT-base (36 fusion passes across 12 transformer layers).

### 🔐 [CredChain](https://github.com/MohamedFarouk-1/credential-verification)
Tamper-proof credential verification anchored on-chain. SHA-256 hashes of credentials written to a Solidity registry — sensitive data stays off-chain, verification is instant and permissionless. Deployed on Base Sepolia.
